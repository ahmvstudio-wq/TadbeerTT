## 1.4 First 9-Grid Concept

The first 9 posts create a cohesive visual mosaic when viewed on the profile grid. The color rhythm alternates to create a "breathing" pattern:

```
┌──────────────┬──────────────┬──────────────┐
│  1. TEAL bg  │  2. PHOTO    │  3. CREAM bg │
│  Quote card  │  Reel thumb  │  Carousel    │
├──────────────┼──────────────┼──────────────┤
│  4. PHOTO    │  5. GOLD bg  │  6. PHOTO    │
│  Case study  │  Stats card  │  Reel thumb  │
├──────────────┼──────────────┼──────────────┤
│  7. CREAM bg │  8. PHOTO    │  9. TEAL bg  │
│  Carousel    │  Team/BTS    │  Manifesto   │
└──────────────┴──────────────┴──────────────┘
```

**Color Pattern Logic:**
- Corners (1, 3, 7, 9): Solid backgrounds — creating a visual "frame"
- Edges (2, 4, 6, 8): Photography-based — adding life and texture
- Center (5): Gold accent — the visual anchor of the entire grid

This creates a diagonal symmetry (teal corners ↔ cream corners) with the gold center acting as a brand beacon.
